❖ Konstruktor koji se može pozvati bez stvarnih argumenata, što znači da ili nema
nijedan parametar, ili svi parametri imaju podrazumevane vrednosti argumenata,
naziva se podrazumevani konstruktor (default constructor)
❖ Ovaj konstruktor poziva se pri podrazumevanoj inicijalizaciji, tj. kada se za
inicijalizaciju nekog objekta ne navede eksplicitan inicijalizator, ili se navede prazan
inicijalizator. Na primer:
struct X {
  X (int=0);
  …
};
struct Y {
  X x1, x2;
  Y () : x2() {}
  …
};
int main () {
  X x;
}
